O que é SWEBOK

  • Guia da Engenharia de Software
  • Organizado pela IEEE Computer Society
  • Base de referência para boas práticas

Definição

O SWEBOK é um manual que organiza os principais conhecimentos da Engenharia de Software em áreas de conhecimento.

Importância

Ele padroniza conceitos e serve como apoio para estudo, prática profissional e desenvolvimento de software com qualidade.
O guia contribui para consolidar práticas reconhecidas e preparar estudantes e profissionais para novas tecnologias.
